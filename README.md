# C++ Programming

COP-2224C coursework from Valencia College - Computer Programming and Analysis program.

## Course Overview
Programming in C++, building on C fundamentals to introduce object-oriented
programming, the standard template library, and more advanced data handling.

## Topics Covered
* Object-oriented programming (classes, objects, encapsulation)
* Constructors, destructors, and operator overloading
* Inheritance and polymorphism
* Templates and the Standard Template Library (STL)
* File handling and stream I/O
* Pointers, references, and memory management
* Exception handling

## Skills Developed
* Object-oriented design in C++
* Data structure implementation
* Working with the STL
* Robust input handling and file processing

## Projects and Assignments
Each `UNITx_PROGRAM/` directory contains the assignment for that unit of the
course, demonstrating the concepts introduced in that module.
